pgnet-mappy
===========

Simple script to extract EXIF metadata for import into Google Fusion Tables.

Written for the Python Scripting for Digital Forensics Session at the PGNET 2014 ECENTRE workshop.

Usage
-----------

    mappy.py SOURCE TARGET

Where the source folder contains a set of EXIF-tagged photographs, and the target is a CSV file that will contain the extracted and sorted metadata.
